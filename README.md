# 大召唤术 / astrbot_plugin_summon

群内发送关键词自动艾特指定成员。支持多群多关键词、自定义文案、权限控制、命令别名。

## 功能

- 在群内输入关键词（如"开黑"）自动艾特预设成员
- 关键词**精确匹配**，不会模糊触发
- 每个群独立配置关键词和成员列表
- 可设置自定义文案（如"大召唤术 @张三 @李四"）
- `add` 命令支持内联文案：`/at add 开黑 快来 @张三`
- 已有关键词再次 `add` 会更新成员，保留原自定义文案（除非指定新文案）
- 权限开关：所有人可触发 / 仅管理员可触发
- 命令别名：`/at` 和 `/大召唤术` 均可使用，支持自定义更多别名
- 所有规则通过群内命令实时管理

## 命令

| 命令 | 说明 |
|------|------|
| `/at add <关键词> [文案] @成员...` | 添加/更新召唤规则，文案可选 |
| `/at del <关键词>` | 删除召唤规则 |
| `/at list` | 查看当前群所有规则 |
| `/at msg <关键词> [文案]` | 单独修改文案（不写恢复默认） |
| `/at delmsg <关键词>` | 清除自定义文案，恢复默认 |
| `/at toggle` | 切换当前群权限模式（仅管理员） |
| `/at help` | 显示帮助 |

> 所有命令均可用 `/大召唤术` 代替 `/at`，例如 `/大召唤术 add 开黑 @张三`。

### add 命令说明

| 输入 | 关键词 | 文案 | 成员 |
|------|--------|------|------|
| `/at add 开黑 @张三` | 开黑 | 召唤（默认） | 张三 |
| `/at add 开黑 快来 @张三 @李四` | 开黑 | 快来 | 张三、李四 |
| `/at add 开黑 快来吧朋友们 @张三` | 开黑 | 快来吧朋友们 | 张三 |

对已有关键词再次 `add` 会更新成员列表，**保留原自定义文案**，除非你指定了新文案。

## 安装

将 `astrbot_plugin_summon` 文件夹放到 AstrBot 的 `data/plugins/` 目录下，然后在 WebUI 插件管理页面重载插件。

## 配置

WebUI 插件配置页面可设置以下选项：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `permission_mode` | `everyone` | 默认权限模式：`everyone`=所有人可触发和管理规则，`admin_only`=仅管理员 |
| `default_message` | `召唤` | 触发关键词时的默认前置文案 |
| `command_prefixes` | `["at", "大召唤术"]` | 命令前缀列表，支持自定义多个别名 |

每个群可用 `/at toggle` 单独切换权限模式。

## 权限说明

| 子命令 | everyone 模式 | admin_only 模式 |
|--------|:---:|:---:|
| `help` | 所有人 | 所有人 |
| `list` | 所有人 | 所有人 |
| `add` | 所有人 | 仅管理员 |
| `del` | 所有人 | 仅管理员 |
| `msg` | 所有人 | 仅管理员 |
| `delmsg` | 所有人 | 仅管理员 |
| `toggle` | 仅管理员 | 仅管理员 |

## 数据持久化

规则存储在 `data/plugins/astrbot_plugin_summon/rules.json`，不会写入插件源目录。

## 适配器

- aiocqhttp（NapCat / go-cqhttp）
